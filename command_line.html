Command Line

I've had some exposure to the command line before for my other classes, so it's not strange to me although I'm not an expert either. Learning the commands is actually fun. At first, I thought it was overwhelming but navigating through my local repositories was not so bad. You will find yourself repeating many of the same commands which helps learning them even faster. I've also done the workflow such as the commits, and push to Github. Those get a bit more tricky, but still nothing to be nervous about.
